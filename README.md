# LostFound-Hub

A simple and responsive Lost & Found web application that helps students report, discover, and recover lost or found items within the campus. The platform provides an easy-to-use interface for posting item details, managing listings, and connecting users through a secure login system.

---

## 📌 Overview

The  LostFound-Hub is designed to simplify the process of reporting lost and found items in a college campus. Users can create an account, log in securely, post details of lost or found belongings, browse available listings, and manage their own posts.

This project focuses on building a clean, responsive, and user-friendly interface using modern web technologies.

---

## ✨ Features

- User Registration & Login
- Secure Authentication
- Post Lost Items
- Post Found Items
- Browse All Listings
- Search Items
- View Item Details
- Edit & Delete Own Posts
- User Profile Management
- Responsive Design
- Clean & Modern UI
